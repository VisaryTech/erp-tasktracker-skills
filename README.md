# ERP TaskTracker Skills

Набор скиллов для работы с ERP TaskTracker.

## Требования

- Python 3.10+
- Доступ к ERP API
- .env
```
erp_base_url=
erp_client_id=
erp_client_secret=
projectId=
erp_label_to_approve=
erp_label_approved=
erp_label_to_define=
```

`erp_client_id` должен принадлежать пользователю ERP, который добавлен в проект с ролью `Разработчик`.
