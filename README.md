# ERP TaskTracker Skills

Набор локальных Codex-скиллов для работы с ERP TaskTracker.

Репозиторий включает скиллы для:

- чтения данных задачи;
- публикации комментариев в задачу;
- проверки готовности задачи к разработке.

## Требования

- Python 3.10+
- Доступ к ERP API
- Учетные данные:
  - `.env` с `erp_client_id` и `erp_client_secret`
  - или переменные окружения `erp_client_id` и `erp_client_secret`
